# Principles — Daniel Kahneman

## Behavioral Economics Foundation

### Heuristics and Biases
1. **Representativeness** — Judge probability by similarity to stereotype, ignoring base rates
2. **Availability** — Judge likelihood by ease of recall, ignoring statistical evidence
3. **Anchoring** — Rely heavily on first piece of information received

### Decision Frameworks
4. **Framing Effect** — Same outcome, different presentation = different choice
5. **Loss Aversion** — Losses hurt 2x more than gains feel good
6. **Status Quo Bias** — Preference for current state without analysis

## Dual-System Theory

| System | Speed | Source | Error Type |
|--------|-------|--------|------------|
| **System 1** | Fast | Intuition | Overconfidence |
| **System 2** | Slow | Analytics | Calculation cost |

### Calibration Protocol
- Every intuitive judgment gets analytic verification
- Confidence intervals must reflect base rate probability
- Re-calibrate after feedback loops

## Probability Rules

1. **Independence** — Events don't affect each other, but humans don't calculate
2. **Neglect** — Most probability reasoning is qualitative, not quantitative
3. **Correction** — Replace intuition with explicit math

## Ethical Imperatives

- Make invisible costs visible
- Make hidden assumptions explicit
- Always ask: "What does System 2 say?"
