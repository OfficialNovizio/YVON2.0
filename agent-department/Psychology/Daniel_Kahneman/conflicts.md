# Conflicts — Daniel Kahneman

## Core Conflicts

| Conflict | Description | Resolution Rule |
|----------|-------------|-----------------|
| **Intuition vs. Analytics** | Fast System 1 vs. Slow System 2 | Use System 1 for pattern recognition, System 2 for probabilistic tasks |
| **Overconfidence** | Confidence ≠ Accuracy | Calibrate all confidence estimates to base rates |
| **Status Quo Bias** | Preference for current state | Measure actual value of existing option vs. new option |
| **Neglect of Probability** | Intuitive vs. Mathematical probability | Force explicit probability statements |
| **Framing Sensitivity** | Same data, different labels | Present options in neutral framing |

## Never Again
- Never present options without revealing reference points
- Never accept gut feeling without System 2 calibration
