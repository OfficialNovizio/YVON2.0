## Purpose

Buffett doesn't diversify by default — he concentrates capital on the highest-conviction opportunity. Equal budget allocation between two ventures with unequal ROI is not neutral. It is a choice to underinvest in what's working and keep alive what isn't.

Felix applies this doctrine every time budget is discussed across ventures or channels.

---

## When It Runs

- Any cross-venture budget discussion
- Quarterly budget reviews
- When Marcus asks how to split budget between Novizio and Hourbour
- When one venture is significantly outperforming the other on ROI signals
- When a channel is clearly outperforming others within a single venture

---

## The Allocation Protocol

### Step 1 — Pull comparative ROI signals
Compare ventures on the five core signals:

| Signal | Novizio | Hourbour |
|--------|---------|----------|
| ROAS / LTV:CAC | | |
| CAC payback period | | |
| MoM growth rate | | |
| Gross margin | | |
| Runway consumption per dollar | | |

### Step 2 — Concentration test
**Is one venture clearly outperforming on ≥ 3 of 5 signals?**

- **YES → Recommend concentration.** State explicitly: "The data supports increasing [venture] budget by $X and reducing [venture] budget by $X. Rationale: [which signals and by how much]."
- **NO → Both ventures are competitive.** State explicitly: "Equal or proportional split is defensible. Rerun this test in [timeframe] as signals develop."

### Step 3 — Name the reallocation trigger
What specific metric change would cause the current allocation to shift? Pre-name it before Marcus acts on the recommendation.

Examples:
- "If Hourbour's LTV:CAC drops below 3:1 for 2 consecutive months → reallocate to Novizio."
- "If Novizio ROAS exceeds 4x for a full quarter → increase paid acquisition budget by 30%."

Pre-named triggers prevent reactive allocation decisions made under pressure.

### Step 4 — Present the trade-off to Marcus
Felix recommends. Marcus decides. Felix never makes the cross-venture priority call unilaterally.

```
Concentration signal: [venture] outperforms on [N] of 5 signals
Recommendation: Increase [venture] budget $X / Reduce [venture] budget $X
What we sacrifice: [what the underinvested venture loses]
Reallocation trigger: [metric + threshold + timeframe that reverses this]
Decision required from: Marcus
```
