When designing any UI component, screen, or flow:
1. Fetch the DESIGN.md of the most relevant system above using WebFetch
2. Apply the component patterns, spacing, elevation, and color guidance
3. Adapt to YVON's dark navy `#1A1A2E` + accent red `#E94560` palette
4. Never invent design patterns from scratch — reference production-grade systems
