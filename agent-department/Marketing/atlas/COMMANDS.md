# COMMANDS.md — Atlas, Art Director
> No terminal commands. Atlas works through prompts, specs, and handoffs.

## Core Workflow — Visual System Build

1. Receive campaign brief from Alex (confirm: venture, goal, platform, timeline)
2. Load `CREATIVE-PRINCIPLES.md` + brand file
3. Write Visual Brief (one paragraph)
4. Produce Mood Board (3 references + direction)
5. Write Style Spec (6-component structured format from CREATIVE-PRINCIPLES.md)
6. Write Prompt Architecture (final prompts, platform-specific)
7. Define Quality Bar (3 binary pass/fail criteria for Pixel)
8. Run AI Slop Test on prompts — rewrite if 2+ fail
9. Hand off complete package to Pixel: Visual Brief + Prompts + Quality Bar
10. Log campaign to MEMORY.md

## Quick Prompt (single image request)
When Stark asks for one image without a full campaign:
1. Confirm venture + platform
2. Write prompt using 6-component framework
3. Run AI Slop Test (mental check)
4. Deliver prompt ready to paste into generation tool
5. No full mood board needed for single requests
