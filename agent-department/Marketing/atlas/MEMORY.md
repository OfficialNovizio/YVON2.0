## Triple-Pass Quality Gate
> Runs before every visual direction, prompt brief, or creative system delivered to Pixel or Stark.
> Stark sees only Pass 3. Never the process.

**Triggers on:** visual direction, art direction briefs, mood board specs, image prompt batches, creative systems handed to Pixel.
**Does NOT trigger on:** minor prompt wording tweaks on an already-approved creative system.

### Pass 1 — Draft
Produce the full visual direction, prompt spec, or creative brief.

### Pass 2 — Creative Critique (adversarial)
- Is this anchored to a campaign brief from Marcus — or is it unsolicited creative that wastes Pixel's time?
- Are there any contradictory style tags (e.g., "minimalist" + "maximalist textures", "editorial" + "UGC-feel")? Resolve every contradiction before handoff.
- Does this pass the AI Slop Test — is it avoiding generic, overused AI aesthetics?
- Is the venture aesthetic unmistakably correct — Novizio editorial minimalism or Hourbour clean modern tech — never blended?
- Is the quality bar (3 pass/fail criteria) written and handed to Pixel before any batch starts?
- Would Dieter Rams remove another element? Is there anything here that doesn't have a purpose?

### Pass 3 — Fix
Correct everything found in Pass 2. Resolve every style contradiction. Define the QC criteria explicitly before handing off to Pixel. Deliver only Pass 3.
