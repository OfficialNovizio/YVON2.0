## Venture Context
- **Aesthetic**: editorial minimalism — magazine spread, not catalogue
- **Positioning**: premium DTC fashion; clothes that carry meaning

## Novizio Visual System
| Element | Rule |
|---------|------|
| Lighting | Natural, soft, directional — never harsh flash or studio over-lit |
| Background | Clean, neutral, textured walls — never busy patterns or branded backdrops |
| Models/People | Confident, authentic — not posed or commercial-looking |
| Colour palette | Warm neutrals, muted tones — never loud/clashing |
| Composition | Negative space allowed; rule of thirds; avoid centred catalogue feel |
| Style | Editorial lifestyle, street-contextual — never e-commerce flat lay |

## What to Always Avoid (Novizio)
- Stock-photo energy: overly posed, forced smiles
- Promotional overlays: text burns, price tags, "NEW IN" banners
- Busy patterns in backgrounds that compete with the garment
- Oversaturated or high-contrast processed looks
- Seasonal clichés (Valentine's roses, Christmas tinsel)

## Prompt Architecture Notes (Novizio)
- Lead prompt with: lighting description → model direction → background → mood adjectives
- Avoid "professional photography" as a tag — produces stock-photo feel
- Use: "35mm film grain", "natural light", "candid editorial" for texture
- Test: "Paris street photography aesthetic" for instant Novizio register
