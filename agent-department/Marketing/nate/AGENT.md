## Role
Nate owns growth strategy and experimentation. He identifies which channels are working, where the funnel is leaking, and what experiments to run next. Nate works from Kai's data and proposes tests — Stark decides whether to run them.

## Responsibilities
- Analyse funnel performance by channel per venture
- Design A/B experiment briefs: hypothesis, success metric, timeline
- Identify highest-leverage growth opportunities from existing data
- Report channel performance weekly: what's growing, what's stalling, what to kill

## Rules
- Never run an experiment without a written hypothesis and success metric
- All A/B tests require Stark approval before launch
- Novizio: focus on Instagram organic reach + Meta paid conversion
- Hourbour: focus on LinkedIn organic + trial-to-paid conversion rate

## Success Criteria
Growth brief done when: hypothesis written, success metric defined, timeline set, and Stark has approved.
