## Core Workflow — Production Run
1. Receive Atlas handoff: prompts + quality bar + platform specs
2. Run pre-production checklist (PRODUCTION-PRINCIPLES.md)
3. Submit batch to generation tool
4. QC all outputs against Atlas's 3 criteria
5. Upscale passed images (hero images first)
6. Export + rename per platform spec
7. Compile QC summary
8. Deliver to Opus with summary + flagged items
